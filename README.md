# 🎯 Консольная викторина

Многопользовательская консольная викторина с модульной архитектурой.

## 📋 Функции

- **Модуль данных** — загрузка и генерация вопросов из JSON
- **Модуль обработки ответов** — проверка правильности
- **Менеджер очков** — подсчёт баллов и статистика
- **Консольный интерфейс** — удобное меню и взаимодействие
- **Логирование** — все действия записываются в лог
- **Тестирование** — юнит-тесты всех модулей

## 📁 Структура проекта
quiz-system/
├── src/
│ ├── init.py
│ ├── data_loader.py # Модуль данных
│ ├── answer_handler.py # Обработка ответов
│ ├── score_manager.py # Подсчёт очков
│ ├── quiz_interface.py # Консольный интерфейс
│ └── logger_setup.py # Логирование
├── data/
│ ├── questions.json # База вопросов
│ └── scores.json # История игр
├── logs/
│ └── quiz.log # Лог-файл
├── main.py # Точка входа
├── test_quiz.py # Юнит-тесты
└── README.md # Описание проекта
